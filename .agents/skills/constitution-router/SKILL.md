# 🧭 ENRUTADOR CONSTITUCIONAL DIAMOND v12

Este skill funciona como un índice maestro para que sepas exactamente qué archivo constitucional debes leer o invocar según la tarea que se te ha asignado. **NO adivines**, guíate por este mapa.

## 🎯 MAPA DE DECISIÓN DE SKILLS

Analiza tu tarea actual y aplica el Skill correspondiente:

### 1. `/constitution-master` (El Núcleo)
**Cuándo usarlo:** Al iniciar cualquier tarea nueva o cuando necesites recordar cómo operar en general.
**Qué contiene:** El ciclo de vida obligatorio de una tarea (Deconstrucción, Handshake, Diff Atómico, Certificación, Purga).

### 2. `/constitution-core` (Frontend y Estética)
**Cuándo usarlo:** Si tu tarea involucra crear o modificar **UI (Frontend)**, diseñar componentes, o si vas a declarar/importar **tipos de base de datos**.
**Qué contiene:** Leyes de estética (Diseño Luminous, `borderRadius: 40`, LOOP Green `#22c55e`), prohibición de lógica pesada en el cliente, y el mandato de Tipado SSOT.

### 3. `/constitution-webhooks` (Backend y Meta API)
**Cuándo usarlo:** Si tu tarea involucra modificar APIs de **Backend, Webhooks de WhatsApp, Edge Functions, Manejo de Órdenes, Catálogos o Supabase**.
**Qué contiene:** Reglas de Aislamiento Tenant (paso del `company_id`), reglas "Cero 'any'", telemetría Platinum (`logEvent`), y estándares de Catálogos de Meta (Regla del guión bajo).

### 4. `/constitution-hygiene` (Mantenimiento de Memoria)
**Cuándo usarlo:** Al finalizar una tarea grande, si generaste archivos temporales, o si notas que el contexto de NotebookLM (el Oráculo) está alucinando código viejo.
**Qué contiene:** El mandato estricto de purgar reportes temporales mediante los MCP tools y el bloqueo de seguridad si el Oráculo pierde conexión.

Identifica tu tarea actual y procede INMEDIATAMENTE a aplicar el mandato del Skill específico que le corresponda.
